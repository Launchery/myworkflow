# Workflow Command Index

## Stage Commands

1. `/wf.discover` - Stage 1 discovery
2. `/wf.spike` - Stage 2 technical spike
3. `/wf.arch` - Stage 3 architecture (HR)
4. `/wf.spec` - Stage 4 specification (HR)
5. `/wf.plan` - Stage 5 implementation plan (HR)
6. `/wf.tasks` - Stage 6 task passports (HR)
7. `/wf.tooling` - Stage 7 tooling workflow (HR)
8. `/wf.dispatch` - Stage 8 dispatch plan
9. `/wf.implement` - Stage 9 implementation
10. `/wf.review` - Stage 10 review
11. `/wf.finish-branch` - Stage 11 branch finalization
12. `/wf.project-report` - Stage 12 project report
13. `/wf.human-qa` - Stage 13 human QA
14. `/wf.debug` - Stage 14 debug loop
15. `/wf.finish-report` - Stage 15 final report

## Service Commands

1. `/wf.status` - Show workflow status
2. `/wf.resume` - Resume interrupted stage
3. `/wf.gates` - Show gate results
4. `/wf.history` - Show workflow history
5. `/wf.approve <stage>` - Approve stage
6. `/wf.reject <stage>` - Reject stage
